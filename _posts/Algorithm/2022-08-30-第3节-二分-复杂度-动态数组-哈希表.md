### 有序数组中找到num

```java
	// arr保证从小到大有序
	public static boolean find(int[] arr, int num) {
		if (arr == null || arr.length == 0) {
			return false;
		}
        // 最开始是在整个范围上找值,所以取的是左右最大值
		int L = 0;
		int R = arr.length - 1;
        // L <= R是有效的范围,一旦 L > R 无效,不再进行后续处理
		while (L <= R) {
			int mid = (L + R) / 2;
			if (arr[mid] == num) {
				return true;
			} else if (arr[mid] < num) {
				L = mid + 1;
			} else {
				R = mid - 1;
			}
		}
		return false;
	}
```

有序数组中找到>=num最左的位置

```java
    // 最左就是二分不断进行处理,而不是找到一个值相等就不进行剩余操作
	// arr有序的，>=num 最左
	public static int mostLeftNoLessNumIndex(int[] arr, int num) {
		if (arr == null || arr.length == 0) {
			return -1;
		}
		int L = 0;
		int R = arr.length - 1;
        // 记录里面最小的值为-1
		int ans = -1;
		while (L <= R) {
            // 取结果的整数,去掉余数
			int mid = (L + R) / 2;
            // 这个和上面的区别就是不再是找某个具体值,而是对比值的情况
			if (arr[mid] >= num) {
				ans = mid;
				R = mid - 1;
			} else {
				L = mid + 1;
			}
		}
		return ans;
	}
```

有序数组中找到<=num最右的位置

```java
	// 在arr上，找满足<=value的最右位置
	public static int nearestIndex(int[] arr, int value) {
		int L = 0;
		int R = arr.length - 1;
		int index = -1; // 记录最右的对号
		while (L <= R) {
			int mid = L + ((R - L) >> 1);
			if (arr[mid] <= value) {
				index = mid;
				L = mid + 1;
			} else {
				R = mid - 1;
			}
		}
		return index;
	}
```

局部最小值问题
```java
//  arr 无序 任意两个相邻的数不相等！
// 局部最小定义
        // [0] < [1] 此时0称为局部最小,因为0的左边没有值了
        // [N - 2] > [N - 1] 此时N-1称为局部最小,因为右边没值了
        // 左 > [i] < 右 此时i位置称为局部最小,因为左右都比i值大

// 本题要求返回任意一个局部最小值
	public static int oneMinIndex(int[] arr) {
        // 此时没有数
		if (arr == null || arr.length == 0) {
			return -1;
		}
		int N = arr.length;
		if (N == 1) {
			return 0;
		}
        // 下面这几行同时也处理了只有两三个值的情况
		if (arr[0] < arr[1]) {
			return 0;
		}
		if (arr[N - 1] < arr[N - 2]) {
			return N - 1;
		}
		int L = 0;
		int R = N - 1;
        // 从之前走到这一步说明
        // 0的位置一定比1的位置大
        // n-1的位置一定比n-2的位置大
        // 此时两边都是局部最大,中间一定存在局部最小值
		// L...R 肯定有局部最小
        // 直接从中间那个点去看,存在四种可能
        // 左 > i > 右 I i . 砍掉左边,往右边找
        // 左 < i < 右 . i I 砍掉右边,往左边找
        // 左 > i < 右 I i I 此时的i就是局部最小
        // 左 < i > 右 . i . 此时两边都大于i 完全抛弃右边,只看左边的值
        // 只要比左边大,就把右边砍掉,因为题目只让返回一个
 		while (L < R - 1) {
            // 注意这个边界是R - 1 和L <= R有着不一样的情况
            // 我们希望算出的mid - 1和 mid +1的值还在L,R范围内
            // 但有时只有L,R两个数,此时算出的mid值为L等于0,这时再去算mid-1就会导致小于L,超过边界
            // 此时改为R - 1,保证一定右三个数,两个数的情况下再接着下一步判断处理
			int mid = (L + R) / 2;
            // 同时小
			if (arr[mid] < arr[mid - 1] && arr[mid] < arr[mid + 1]) {
				return mid;
			} else {
                // 并不同时小,此时有三种情况
                // 1左>我 我>右 I i .
                // 2左<我 我<右 . i I 
                // 3左<我 我>右 . i .

                // 中间大于左侧 . i 对应两种情况 2,3,此时只看左侧
				if (arr[mid] > arr[mid - 1]) {
					R = mid - 1;
				} else {
                    // 走到这步说明.中间小于等于 左边的值 I i 
                    // 对应情况1
					L = mid + 1;
				}
			}
		}
		return arr[L] < arr[R] ? L : R;
	}
```
